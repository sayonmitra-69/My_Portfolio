# 🚀 Sayon Mitra - Developer Portfolio

A modern and responsive full-stack developer portfolio built using React.js, Tailwind CSS, Framer Motion, Node.js, and Express.js.

Designed with a premium dark UI aesthetic, smooth animations, responsive layouts, and interactive sections to showcase projects, skills, resume, and contact functionality professionally.

---

# 🌐 Features

- ⚡ Modern responsive UI
- 🎨 Tailwind CSS styling
- ✨ Framer Motion animations
- 📱 Mobile responsive design
- 🧩 Reusable React components
- 📂 Interactive projects showcase
- 🛠️ Animated skills section
- 📄 Resume download integration
- 📬 Express.js backend contact API
- 🌙 Premium dark theme
- 🚀 Smooth scrolling navigation

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Framer Motion
- JavaScript
- React Icons
- Axios

## Backend
- Node.js
- Express.js
- REST APIs
